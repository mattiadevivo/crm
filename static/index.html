CRM backend

  • GET http://localhost:3000/customers - Get all customers
  • GET http://localhost:3000/customers/:id - Get customer with the specified id
  • POST http://localhost:3000/customers - Stored inside db the customer provided via body (json)
  • PUT http://localhost:3000/customers/:id - Update customer having the specified id in db with data provided via body (json)
  • DELETE http://localhost:3000/customers/:id - Delete customer with the specified id
